Erhebungszeitraum 

die Daten stammen aus dem Jahr - данные originate из ... года
die Statistik wurde im Jahre ... gesammelt - статистика была собрана
die Studie wurde im Jahre ... durchgeführt - исследование было проведено в ... году

Beispiel :: die Daten basieren auf einer (repräsentative) Umfrage, 
die im Zeitraum vom ... bis ... durchgeführt wurde.

Пример :: данные базируются на репрезентативном опроме, 
который был провдеен в промежуток с .. по .. года.
